这是博客系统的前端页面部分

前端部分共有以下网页
    1. index.html     //登录页面，首页
    2. regerror.html   //因为恶意修改前端代码而导致注册失败的返回页
    3. newwelcome.jsp   //主页，展现所有人的动态
    4. my_newwelcome.jsp    //个人主页，展现自己的动态
    5. score.jsp        //打分页面
    6. scoredetails.jsp     //打分之后的展示页面
    7. adminlog.jsp     //管理员登录页
    8. admin.jsp        //管理员主页
    9. showscore.jsp        //查看具体某个学生分数的页面
